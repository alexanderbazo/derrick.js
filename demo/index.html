Derrick.js
derrick.js is a simple JavaScript library to define and detect custom [time framed]Â event chains. Derrick uses a simple syntax to define event patterns and to add listeners to them. Triggered events are automatically processed. When Derrick detects a predefined pattern, registerd listeners are envoked. More pattern constraints will be added soon. To use Derrick you will need the library and the web worker script. The script may be used under MIT License.

Live-Demo (pattern: red, green, blue [1s])

Include

        <script src="derrick.js"></script>

Usage (for demo pattern)

        // Start Derrick with webworker script
        Derrick.start('/js/src/derrick_worker.js'); 
        
        // Add pattern, callback [and timeframe]
        Derrick.addPatternListener('rgb-sequence', ['red', 'green', 'blue'], function (event) {
            alert("Pattern detected: " + event.pattern);
        }, 1000);
        
        // Trigger your events somewhere
        Derrick.trigger('red');
        
        // You can link derrick with dom events
        document.querySelector('#buttons .green').addTriggerLink('click', 'green');
        document.querySelectorAll('#buttons .box').addTriggerLink('click', 'box');
